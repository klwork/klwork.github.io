Linux常用命令和技巧

Linux 2016/05/06

本文于1017天之前发表，文中内容可能已经过时。

Linux常用命令

使用lsof查看占用端口的进程

lsof -i :3306

或者使用

sudo netstat -anop | grep 3306

-p 选项查看进程信息

netstat -tnlp

可以查看监听的端口，其中l是listening, p是显示program, n是显示ip而不是name, t只查看tcp的内容

查看发包的端口

netstat -tu -c

-t 选项列出 TCP 协议的连接
-u 显示UDP协议的连接情况
-c 每隔1秒就重新显示一遍，直到用户中断它

显示某个程序是否在运行，查看某个运行的程序

ps -aux | grep "memcached"

设置DNS、网关

echo “nameserver 202.16.53.68″ >> /etc/resolv.conf
route add default gw 192.168.1.1

删除所有空目录

find /data -type d -empty -exec rm -rf {} \;

linux下批量删除空文件（大小等于0的文件）的方法

find /data -type f -size 0c -exec rm -rf {} \;
find /data -type f -size 0c|xargs rm –f

删除五天前的文件

find /data -mtime +5 -type f -exec rm -rf {} \;

删除两个文件重复的部份，打印其它

cat 1.txt 3.txt |sort |uniq

查看系统版本

lsb_release -a

如何查看目标主机192.168.0.1开放那些端口

yum install nmap
nmap -PS 192.168.0.1
