#Shell脚本的基本规则

start

在 Shell 脚本的第 1 行开始处指定“#!/bin/sh”

双引号中包含的变量可以用其值来替换，而如果是单引号，则依然保持变量名。

当需要将变量名和其他字符串连接在一起时，请用如下的 {} 符号将变量名括起来
echo “${message}World”

Shell脚本运行时的参数的引用

可以通过变量 $1~$9 来访问 Shell 脚本运行时的参数。用于启动 Shell 脚本的命令的名称本身也可以通过 $0 访问。

“$@”为一个包含所有参数的列表
