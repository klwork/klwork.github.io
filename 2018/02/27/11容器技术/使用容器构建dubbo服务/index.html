使用容器构建dubbo服务

容器技术 中间件 分布式 Docker 2018/02/27

背景

安装步骤

假设运行docker的机器为10.168.1.111

创建Tomcat容器

docker pull chaimm/tomcat:1.1
docker run --name gaoxi-user-1 -p 8082:8080 -v /mydata/temp/tomcat/log:/opt/tomcat/gaoxi-log chaimm/tomcat:1.1

–name：指定容器的名字

-p：指定容器的端口映射

-p 8082:8080 表示将容器的8080端口映射到宿主机的8082端口上

-v：指定容器数据卷的映射

xxx:yyy 表示将容器yyy目录映射到宿主机的xxx目录上，从而访问宿主机的xxx目录就相当于访问容器的yyy目录。

chaimm/tomcat:1.1：表示容器所对应的镜像。

测试
通过
http://10.168.1.111:8082 访问到gaoxi-user-1容器的tomcat
